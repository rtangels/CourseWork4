Программа сбора и систематизации информации
с сайтов по подбору персонала.

Имеет классы и функуии для работы с API сайтов по подбору персонала. Помогает
получить данные, и произвести с ними необходимые действия
Имеет  функцию для взаимодействия с пользователем. Функция взаимодействует
с пользователем через консоль.
